# ShieldVPN

A modern VPN client application built with SwiftUI for iOS devices. ShieldVPN provides a secure and user-friendly way to connect to VPN servers.

## Features

- List of VPN servers with country flags
- Easy server selection interface
- Connect/Disconnect functionality
- Modern SwiftUI interface
- Server status indicators

## Requirements

- iOS 15.0+
- Xcode 13.0+
- Swift 5.5+

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yusufcanvar/ShieldVPN-Swift-Apple-App.git
```

2. Open `ShieldVPN.xcodeproj` in Xcode

3. Build and run the project
